  ¿QUÉ ES JQUERY?
    - es una biblioteca multiplataforma, creada inicialmente por john resig, que permite simplificar la manera de interactuar con los documentos html, manipular el árbol dom, manejar eventos, desarrollar animaciones y agregar interacción con la técnica ajax en páginas web

    - es una biblioteca de javascript rápida, pequeña y rica en funciones, hace cosas como el recorrido y la manipulación de documentos html, el manejo de eventos, la animación y ajax sean mucho más simples con una api fácil de usar que funciona en una multitud de navegadores, con una combinación de versatilidad y extensibilidad

    - llego a ser con mucho, la biblioteca de javascript más popular del mundo que se encuentra en más del 80 por ciento de los 1 millon de sitios web más importantes del mundo, eso es más de cuatro veces la adopción de angular, react y vue, combinados

    - creado en 2006 por john resig, original mente querian llamarlo jselect pero todos los dominios fueron tomados

    - los navegadores tenian un monto de problemas de interoperabilidad y juqyert proporcionó una api única para manejar los casos extremos en múltiples navegadores, eso fue tam bueno que los proveedores de navegadores de hoy han estandarizado muchas de sus caracteristicas y las envian como api nativas

  ¿QUÉ ES EL DOM?
    - el modelo de objeto de documento (DOM) es una interfaz de programación para los documentos html y xml, faciilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido

  CONCEPTOS
    - JQuery $(this): selector para acceder a elementos hijo de un elemento, nos permite posicionarnos en un determinado elemento e interactuar con los elementos hijo

    - JQuery attr(): establece o devuelve atributos y valores de elementos seleccionados, cuando se usa este método para devolver el valor del atributo, devuelve el valor del primer elemento coincidente 

    - jQuery data(): almacena datos arbitrarios asociados con los elementos coincidentes o devuelve el valor en el almacen de datos designado para el primer elemento del conjunto de elementos coincidentes

    - jQuery on(): adjunta una función de controlador de eventos para uno o más eventos a los elementos seleccionados

    - jQuery Event Delegation: se refiere al proceso de usar propagación de eventos (burbujeo) para manejar eventos en un nivel más alto en el DOM que el elemento en el que se originó el evento, nos permite adjuntar un solo detector de eventos para elementos que existen ahora o en el futuro, dentro de la función de manejo de eventos

    - jQuery filter(): reduce el conjunto de elementos coincidentes a aquellos que coincidan con el selector o pasen la prueba de la funcion

    - jQuery slideToggle(): mostrar u ocultar elementos coincidentes con un movimiento deslizante

    - jQuery preventDefault(): si se llama a este método no se activará la acción predeterminada del evento

    - jQuery css(): obtiene el valor de una propiedad de estilo calculada para el primer elemento en el conjunto de elementos coincidentes o establezca una o más propiedades css para cada elemento coincidente

    - jQuery class attribute: estos metodos inspeccionan y manipulan las clases css asignadas a los elementos

    - jQuery animate: realiza una animacion personalizada de un conjunto de propiedades css

    - jQuery Ajax: realice una solicitud HTTP asincrona, ajax significa javascript asincrono y xml, permite que un usuario de la aplicacion web interactue con una pagina web sin la interrupcion que implica volver a cargar la pagina web, la interaccion del sitio web ocurre rapidamente solo con partes de la pagina de recarga y renovacion 

    - Control de acceso HTTP (CORS): el intercambio de recursos de origen cruzado (CORS) es un mecanismo que utiliza cabeceras HTTP adicionales para permitir que un user agent (en-US) obtenga permiso para acceder a recuersos seleccionados desde un servidor, en un origen distinto (dominio) al que pertenece. Un agente crea una petición HTTP de origen cruzado solicita un recurso desde un dominio distinto, un protocolo o un puerto diferente al del documento que lo generó.

    - Cross-site scripting: cross-site scripting (xss) es una vulnerabilidad de seguridad que permite a un atacante inyectar en un sitio web código malicioso del lado del cliente, este código es ejecutado por las victimas y permite a los atacantes eludir los controles de acceso y hacerse pasar por usuarios, según el open web application security project, xss fue la séptima vulnerabilidad más común de las aplicaciones web en 2017

    - jQuery Plugin: es un metodo que usamos para extender el objeto prototipo de jquery, es un fragmento de codigo escrito en un archivo javascript, que permite que todos los objetos jquery hereden cualquier metodo que agregue

    - jQuery extend(): combinar el contenido de dos o mas objetos juntos en el primer objeto

    - jQuery Deferred(): en jQuery es una funcion que devuelve el objeto de utilidad con metodos que pueden registrar multiples devoluciones de llamada a las colas. Llama a las colas de devolucion de llamada y transmite el estado de exito o falla de cualquier funcion sincrona o asincrona

    - jQuery when(): proporciona una forma de ejecutar funciones de devolución de llamada basadas en cero o más objetos de thenable, generalmente objetos diferidos que representan eventos asíncronos
